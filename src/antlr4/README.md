
# ANTLR4

Installs ANTLR4 as binary.

## Example Usage

```json
"features": {
    "ghcr.io/NikiforovAll/devcontainer-features/antlr4:1": {
        "version": "latest"
    }
}
```
## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| version | Select the version to install. | string | latest |
